# 打砖块

这是第二个 Codex 练习游戏。直接用浏览器打开 `block/index.html` 就能玩。

## 玩法

- 空格或“开始 / 暂停”按钮控制游戏开始和暂停
- 鼠标移动、方向键或 A/D 控制弹板
- 生命值用游戏画面左上角的心形图标表示
- 所有球都掉落会损失 1 命
- 清空当前关卡所有砖块后进入下一关
- 通关所有关卡后游戏成功

## 砖块和装备

- 普通砖块有不同血量，血条显示剩余生命
- `?` 是特殊砖块，打碎后随机掉落装备
- 白点橙色砖块是爆炸砖块，打碎后会伤害附近砖块
- `W`：弹板变长一段时间
- `+2`：永久增加 2 个球，直到这些球掉落
- `B`：限时爆炸球，击中砖块会造成范围伤害，但球仍会正常反弹
- `S`：弹板两端限时获得射击能力，单击鼠标发射子弹

## 继续练习 Codex 的提示词

- `请解释 block/game.js 的关卡配置。`
- `请再加 2 个关卡，并让形状更明显。`
- `请把 game.js 拆成 level、render、physics 三个文件。`
- `请给装备掉落加权重，让球数增加更稀有。`
